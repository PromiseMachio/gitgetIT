## 🌍 Deployment

You can deploy these projects using:

- GitHub Pages  
- Netlify  
- Vercel  

### 🚀 Deploy with GitHub Pages

1. Push your code to GitHub  
2. Go to repository **Settings**  
3. Click on **Pages**  
4. Select branch (`main`)  
5. Save and access your live site
